💡 什麼是 Skill？

Skill = AI 的「說明書」

把你今天做的所有東西打包成一份文件：

  • ✅ HTML 模板的結構說明
  • ✅ 5 個 Prompt 組合
  • ✅ 工作流程步驟
  • ✅ 注意事項與規範
好處：以後只要告訴 AI「請看這份 Skill」，AI 就能快速理解你的需求！

沒有 Skill vs 有 Skill

沒有 Skill有 Skill
每次都要重新解釋一次設定，永久使用
結果不穩定品質一致
花時間溝通快速產出

  • examples/：成品範例
  • assets/：素材庫

📝 README.md 怎麼寫？

# HTML 簡報製作 Skill

## 這是什麼？
這是一套用於製作 HTML 簡報的工具組，包含模板和 Prompt。

## 使用方式
1. 複製 template.html 作為新簡報的基礎
2. 使用 prompts/ 資料夾中的 Prompt 生成內容
3. 把生成的內容填入模板的【】標記處

## 模板結構說明
- 封面頁：【簡報標題】【副標題】【講者姓名】
- 內容頁：【章節標題】【重點1-5】
- 結尾頁：【感謝文字】【聯絡資訊】

## Prompt 使用順序
1. 先用「標題生成器」想標題
2. 用「重點條列器」整理每頁重點
3. 用「開場白生成器」寫開場
4. 用「結語生成器」寫結尾

## 注意事項
- 每頁重點不超過 5 個
- 每個重點不超過 20 字
- 配色請遵循模板的預設值

🎯 如何告訴 AI 使用你的 Skill？

方法 1：直接貼 README

幫我製作一份關於「AI 入門」的簡報：

[貼上 README.md 的內容]

現在請幫我生成封面頁的內容。

方法 2：上傳整個資料夾

如果 AI 工具支援上傳檔案，可以直接上傳整個 Skill 資料夾。

方法 3：存成 GitHub Repo

請參考這個 GitHub 專案的 Skill：
https://github.com/你的帳號/html-slides-skill

然後幫我製作一份關於
「時間管理」的簡報。
最推薦：放到 GitHub，這樣隨時都能給 AI 看！
